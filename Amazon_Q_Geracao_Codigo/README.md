Neste laboratorio, o objetivo foi gerar histórias infantis, usei as sugestões de código em linha do Amazon Q Developer para ajudar a completar uma função do AWS Lambda que faz uma chamada de API para um modelo de base do Amazon Bedrock. A função Lambda também armazena a história como um arquivo de texto no Amazon S3 e armazena metadados associados no Amazon DynamoDB. 

- Completei uma função AWS lAMBDA DE CONTADOR DE HISTORIAS com a ajuda do Amazon Q.
- Testei a função Lambda para confirmar que o código está funcionando.
- Revisei os resultados armazenados no Amazon S3
